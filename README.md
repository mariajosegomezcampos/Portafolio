# Portafolio de María José Gómez

¡Bienvenida/o! Este es mi portafolio profesional donde muestro mis proyectos y mi enfoque en arquitecturas Frontend sólidas y escalables.

## 🚀 Tecnologías utilizadas
- **React** (Vite)
- **TypeScript**
- **Styled Components**
- **Framer Motion** (para las animaciones)
- **Tailwind CSS**

## 📂 Estructura del proyecto
- `src/components`: Componentes reutilizables como la Navbar y el Drawer lateral.
- `src/styles`: Configuración de temas (Light/Dark mode).

## 🛠️ Cómo ejecutarlo localmente
1. Clona el repositorio.
2. Entra en la carpeta `my-portfolio`.
3. Instala dependencias: `npm install`
4. Ejecuta el servidor: `npm run dev`
